The Big Idea
A small matrix scans the image
A kernel (3×3 matrix) slides across the image. At each position, it multiplies its values by the pixels underneath, sums them up, and writes the result. Different kernels detect different features: edges, blur, sharpen, patterns.

Why Convolution?

A convolutional layer is a learnable feature detector. Instead of hand-designing edge detectors, a CNN learns thousands of kernels from data. Early layers learn edges, later layers learn textures, then parts (eyes, ears), then whole objects. The same kernel slides everywhere — so a "cat feature" in the top-left is detected the same way as one in the bottom-right.

Modern Usage

CNNs power image classification, medical imaging, and self-driving cars. Even diffusion models (DALL-E, Stable Diffusion) use convolutional U-Nets. Vision Transformers (ViT) replace CNNs with attention, but conv layers are still everywhere.
