The Big Idea
A function is a machine
You put a number in, and a specific number comes out. Every input gives exactly one output. That's it! The graph below is just a picture of every input-output pair drawn as a dot.

Try This!

1. Pick y = x². What happens when x is negative?
2. Try y = |x|. Can the output ever be negative?
3. Look at y = step(x). This is how a neuron "decides" — it outputs 0 or 1. This exact function is used in the Perceptron!

Why This Matters for AI

Neural networks are built from functions stacked together. Each neuron applies a function to its inputs. The activation functions you'll see later (ReLU, Sigmoid) are just functions like the ones above — but chosen because they help networks learn.

Key Vocabulary

Domain = all possible inputs (x values). Range = all possible outputs (y values). For y = x², the domain is all numbers, but the range is only 0 and above (you can't get a negative output by squaring).
Most functions we've seen are continuous — you can draw them without lifting your pen. The step function is discontinuous — it jumps. This matters in AI because smooth functions are easier to optimize (gradient descent needs smoothness).
