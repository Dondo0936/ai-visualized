Key Concepts

The loss function measures how wrong a model's predictions are. Each point on this surface represents a set of model parameters (weights). Height = error. Training a neural network = finding the lowest point. Real networks have millions of dimensions — we show 2 so you can visualize it.
The gradient is a vector pointing in the steepest uphill direction. Its magnitude tells you how steep the slope is. Gradient descent steps in the opposite direction (downhill). When the gradient is near zero, you're at a flat spot — hopefully a minimum, but maybe a saddle point.
SGD follows the raw gradient — simple but slow in curved valleys. Momentum adds inertia (like a heavy ball), building speed in consistent directions. Adam adapts the step size per-parameter and tracks both direction and magnitude. Most modern AI (GPT, DALL-E) uses Adam or AdamW.
Every neural network is trained with gradient descent. The model starts with random weights (a random point on the loss surface) and iteratively adjusts them to reduce error. Modern LLMs use AdamW with learning rate warmup + cosine decay, training for days across thousands of GPUs.
