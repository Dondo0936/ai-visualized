The Big Idea
A loss function rates the model's mistakes
To train a model, we need a single number that says "you were this wrong." Different tasks need different loss functions. Regression (predict a number): MSE, MAE. Classification (predict a class): Cross-Entropy.

Classification Loss

Cross-Entropy (true label = 1)
Cross-entropy measures how much probability the model gave to the correct answer. Low prob = huge loss. High prob = near-zero loss.

Key Differences

MSE squares the error — punishes outliers HARD. Smooth gradient. Default for regression.

MAE takes absolute value — treats all errors equally. More robust to outliers, but gradient is not smooth at zero.

Huber = best of both. Quadratic near zero (smooth), linear far from zero (robust).

Cross-Entropy heavily penalizes being confident AND wrong. Used in every classifier (including GPT for next-token prediction!).

Why It Matters

The choice of loss function decides what "good" means to the model. Change it, and the model learns different things. Research papers often introduce new loss functions to fix specific problems (e.g., focal loss for imbalanced classes).

Related
