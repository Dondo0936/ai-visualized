What temperature does. T < 1 sharpens the distribution toward the argmax — safer, more repetitive. T > 1 flattens it — more creative, more chaotic. T=0 is deterministic (just argmax); T→∞ becomes uniform over surviving tokens. Real LLMs (GPT-4, Claude) ship with T=1 but let you ask for T=0 for code and T=1.5 for poetry.

Entropy vs temperature
H(T) → 0 as T → 0 (argmax), H(T) → log2(V) = — bits as T → ∞ (uniform). The vertical line is the current T.
Entropy measures how "spread out" the distribution is. 0 bits means perfectly confident, log2(V) bits means perfectly uniform over V items. Perplexity 2H maps that to an effective vocabulary size.
