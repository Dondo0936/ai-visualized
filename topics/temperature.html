The Big Idea
Temperature = how "creative" the AI is
When ChatGPT picks the next word, it has raw "scores" (called logits) for every possible word. Softmax turns these into probabilities. Temperature controls how much the model favors the top word vs. exploring alternatives.

Temperature (τ)

Drag to control randomness 1.00

Prompt

Sample Next Word

—
Samples a word using the current probabilities. At low τ, you'll almost always get the same top word. At high τ, each click is unpredictable.

Intuition

At τ = 1.0, the model samples using the raw probabilities. Likely words come up more often, but there's still variety.

In Real LLMs

τ = 0 — Always picks the top word. Deterministic, but repetitive and robotic.
τ = 0.7 — Default for many chatbots. Coherent but not boring.
τ = 1.0 — Balanced — raw model probabilities.
τ > 1.5 — Creative writing. Can go off-rails.
τ = ∞ — Uniform random — complete gibberish.

Formula

pi = exp(zi / τ) / Σ exp(zj / τ)
