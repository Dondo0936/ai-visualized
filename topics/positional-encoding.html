Why positional encoding exists. Self-attention is a permutation-equivariant operation: shuffle the input tokens and the output shuffles too. Without an extra signal, "dog bit man" and "man bit dog" would look identical. Positional encoding adds order info either by adding a vector to each token (sinusoidal, learned), rotating query/key pairs (RoPE), or biasing attention directly (ALiBi).
